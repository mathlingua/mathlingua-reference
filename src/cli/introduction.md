# Introduction

Run `mlg` from the root of a MathLingua collection. The command discovers `.mlg` files, parses them, and applies the operation requested by the subcommand.

Most authoring workflows use two commands:

```text
mlg check
mlg view
```

Use `mlg check` when editing source and wanting fast diagnostics. Use `mlg view` when you want the rendered document in a browser.
