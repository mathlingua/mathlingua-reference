# Introduction

MathLingua is a language for writing mathematical knowledge in a structured, checkable, and renderable form.

A MathLingua collection is made of `.mlg` files. Each file contains top-level items such as definitions, theorems, prose blocks, resources, people, and writing aliases. The source is meant to be readable as text, while `mlg view` renders it as a mathematical document with cards, prose, source flips, and linked definitions.

The most useful way to read this reference is:

1. Start with the [cheatsheet](../cheatsheet.md) for a compact tour.
2. Read [Formulation](../formulation/README.md) for the mathematical syntax inside sections.
3. Read [Structural](../structural/README.md) for top-level items and clause groups.
4. Use [Current Implementation Notes](../current_implementation.md) for the current Rust implementation rules and edge cases.
