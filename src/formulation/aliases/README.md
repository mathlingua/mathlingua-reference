# Aliases

Aliases introduce alternate notation or rendering behavior.

MathLingua currently uses three alias operators:

- `:=>` for expression aliases
- `:->` for specification or capability aliases
- `:~>` for writing aliases

Aliases can appear in `Aliases:`, `Requires:`, `Enables:`, `Documented:`, and top-level `Writing:` contexts, depending on the group kind. The surrounding structural section determines the exact meaning.
