# Labels

Labels identify items inside brackets or nested groups.

```mlg
[\theorem.ordinal.trichotomy]
Theorem:
then: ...
```

Label headers are parsed from dotted parts. Dots are commonly rendered as spaces in fallback display names, with words capitalized for theorem-like and axiom-like cards.

Label headers differ from expression labels. Expression labels use the `[:label:]` syntax inside expressions, while structural label headers appear in square brackets before a group.
