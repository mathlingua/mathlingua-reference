# Text

`Text:` is a top-level prose block.

```mlg
Text: "This paragraph supports **Markdown** and $x \in X$.

It may span multiple source lines."
Id: "23e4567e-e89b-42d3-a456-426614174000"
```

`Text:` renders directly in the page flow, not inside a card. Markdown and LaTeX are interpreted by `mlg view`.
