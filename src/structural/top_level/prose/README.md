# Prose

Top-level prose groups render directly on the page rather than inside cards.

```mlg
Title: <OpenText>
SectionTitle: <OpenText>
SubsectionTitle: <OpenText>
Text: <OpenText>
```

`Section:` and `Subsection:` were replaced by `SectionTitle:` and
`SubsectionTitle:`. `Subsubsection:` is no longer part of the implemented
structural language.

`Text:` supports Markdown and LaTeX in `mlg view`, and quoted text may span
multiple source lines.
