# Adjective

`adjective:` is the required documented text item for `Refines:` entries.

```mlg
Documented:
. adjective: "injective"
```

When a refined type is rendered, the adjective is placed before the base type's rendered name. Multiple refinements render as comma-separated adjectives, such as `injective, surjective function`.

`Refines:` may also contain `written:`, but it may not contain `called:`. The adjective supplies the prose name that would otherwise be carried by `called:`.
